# Gomoku AI Project

## 1. Giới thiệu
Dự án phát triển AI chơi cờ Caro (15x15) đáp ứng yêu cầu môn học Nhập môn AI.
- **Minimax Agent:** Sử dụng giải thuật Minimax, Alpha-Beta Pruning và Heuristic đánh giá mẫu cờ.
- **ML Agent:** Sử dụng Convolutional Neural Network (CNN) huấn luyện theo phương pháp Behavioral Cloning (học từ Minimax).
